## 题目描述

给定一个整数数组，判断是否存在重复元素。

如果存在一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。
```
示例 1:

输入: [1,2,3,1]
输出: true
```
```
示例 2:

输入: [1,2,3,4]
输出: false
```
```
示例 3:

输入: [1,1,1,3,3,4,3,2,4,2]
输出: true
```

## 解题思路

1. 字典计数法

## 解题步骤

1. 新建字典 map，将数字为键，数量为值作为键值对保存在字典中
2. 判断是否存在该值

## 时间复杂度和空间复杂度

+ 时间复杂度：O(n)
+ 空间复杂度：O(n)
