## 题目描述

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

**示例 1：**

![](https://assets.leetcode.com/uploads/2021/02/19/rev1ex1.jpg)
```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```

**示例 2：**
![](https://assets.leetcode.com/uploads/2021/02/19/rev1ex2.jpg)
```
输入：head = [1,2]
输出：[2,1]
```
```
示例 3：

输入：head = []
输出：[]
```

提示：

+ 链表中节点的数目范围是 [0, 5000]
+ -5000 <= Node.val <= 5000

## 解题思路

1、反转链表，即将后一个值的next指向前一个值
2、利用双指针遍历链表

## 解题步骤

1、双指针一前一后遍历链表
2、反转双指针

## 时间复杂度和空间复杂度

+ 时间复杂度O(n)
+ 空间复杂度O(1)
