## 题目描述

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

 

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/10/03/swap_ex1.jpg)
```
输入：head = [1,2,3,4]
输出：[2,1,4,3]
```
```
示例 2：

输入：head = []
输出：[]
```
```
示例 3：

输入：head = [1]
输出：[1]
```

提示：

+ 链表中节点的数目在范围 [0, 100] 内
+ 0 <= Node.val <= 100

## 解题思路

1. 两个节点相互交换
2. 将指针下个节点和下下个节点进行交换

## 解题步骤

1. 创建一个新节点，连接在 head 前，便于后续连接
2. 创建一个指针指向链表头部
3. 创建两个临时变量，分别指向下个节点和下下个节点，然后相互转换，重复上述操作。

## 时间复杂度和空间复杂度

+ 时间复杂度O(n)
+ 空间复杂度O(1)
