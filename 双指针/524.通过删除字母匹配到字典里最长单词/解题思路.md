## 题目描述

给你一个字符串 s 和一个字符串数组 dictionary 作为字典，找出并返回字典中最长的字符串，该字符串可以通过删除 s 中的某些字符得到。

如果答案不止一个，返回长度最长且字典序最小的字符串。如果答案不存在，则返回空字符串。
```
示例 1：

输入：s = "abpcplea", dictionary = ["ale","apple","monkey","plea"]
输出："apple"
```
```
示例 2：

输入：s = "abpcplea", dictionary = ["a","b","c"]
输出："a"
```

提示：

+ 1 <= s.length <= 1000
+ 1 <= dictionary.length <= 1000
+ 1 <= dictionary[i].length <= 1000
+ s 和 dictionary[i] 仅由小写英文字母组成

## 解题思路

1. 排序 + 双指针
2. 先按长度最长且字典序最小的字符串，将数组进行排序
3. 依次与字符串进行匹配，看是否为字符串的子串

## 解题步骤

1. 先按长度最长且字典序最小的字符串，将数组进行排序
2. 遍历排序后的数组，新建两个指针p1，p2，分别指向 0，p1 为数组元素的指针，p2 为字符串 s 的指针
3. 循环判断 p2 是否小于 s 的长度，判断当前元素的每个字符与 s 的每个字符是否相等，相等则两个指针都右移，不相等，则 p2 右移，最后判断 p1 已指向数组元素的末端，是则代表该元素就是这个 s 的子串

## 时间复杂度和空间复杂度

+ 时间复杂度：O(n)
+ 空间复杂度：O(n)
